# Contributing

Beiträge sind willkommen. Bitte beachte:

## Lizenz
Alle Beiträge müssen unter CC BY-SA 4.0 lizenziert sein.

## Prioritäten
1. Aktualisierungen bei neuen MASVS/MASWE/MASTG-Versionen
2. Zusätzliche Secure-Coding-Patterns (Flutter, React Native, .NET MAUI, KMP)
3. Weitere regulatorische Mappings (HIPAA, SOC 2, NIS2)
4. Übersetzungen (EN, FR, ES, JA, ZH)
5. Praxis-Findings und Testberichte (anonymisiert)

## Pull Requests
- Ein PR pro Feature/Fix
- CHANGELOG.md aktualisieren
- OWASP-Attribution bei neuen Inhalten aus OWASP-Quellen beibehalten
